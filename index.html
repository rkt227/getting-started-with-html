Web & HTML

Basics of the web
- The Web
- HTML (Hyper Text Markup Language) - Main document type of the web.
- URLs - Used to reference to, and access, a location on the web.
- HTTP - Request protocol sent to servers. Allows the retrieval of data.
- Web Applications.

HTML is made up of:

- Text content - What you see.
- Markup - What decides how it looks.
- References to other documents.
- Links to other pages.

Markup Tags

The opening tag, contents, and closing tag are all referred to as an element.
Elements can be either inline, which means they show up in the line of text, or blocks, which creates an invisible block around the text.

The p tag, or paragraph tag, is a good example of a block. All text within the p tag is surrounded by an invisible block, which initiates a line break.

The b tag makes the contents bold.
The em tag makes the contents italicized.
The u tag makes the contents underlined.
The a (anchor) tag turns text into links.
The img tag allows an image to be displayed in line.
The br tag adds a line break. This is important, because all whitespace is rendered as a single space, in HTML.
Both the img tag and the br tag are examples of void tags. Void tags do not require a closing tag.
Certain tags use attributes to reference other documents. The anchor tag's href, and the img tag's src, are both examples of attributes.
